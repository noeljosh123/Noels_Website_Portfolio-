# Project Images

Add your project images to this folder with the following filenames:

- `roadry.png` - Image for RoaDry: Flood Monitoring & Route Optimization project
- `kwikslot.jpg` - Image for KwikSlot: Cinema Booking System project  
- `fitcal.jpg` - Image for FITCAL: Health Management App project

## Image Requirements

- **Recommended dimensions**: 800x600 pixels or similar aspect ratio (4:3 or 16:9)
- **File formats**: JPG, PNG, or WebP
- **File size**: Optimize images for web (ideally under 500KB each)

## How to Add Images

1. Place your image files in this `public/images/` folder
2. Name them exactly as listed above (e.g., `roadry.jpg`)
3. The images will automatically appear on your website

## Supported Formats

- `.jpg` / `.jpeg`
- `.png`
- `.webp`

Note: If you want to use different filenames, update the `image` property in `constants.ts` for each project.
